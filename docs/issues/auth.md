# AUTH - سامانه احراز هویت و مدیریت حساب

## خلاصه
پیاده‌سازی جریان کامل ثبت‌نام، ورود، مدیریت نشست، و بازیابی رمز عبور برای کاربران Mafia Home. این بخش زیربنای ایمن برای سایر ویژگی‌های آنلاین فراهم می‌کند.

## معیارهای پذیرش
- [ ] کاربر می‌تواند با ایمیل و رمز عبور ثبت‌نام کند و ایمیل تأیید دریافت کند.
- [ ] ورود و خروج با احراز هویت مبتنی بر توکن (مانند JWT) به‌طور ایمن انجام می‌شود.
- [ ] مسیر بازیابی رمز عبور با ارسال لینک به ایمیل کاربر در دسترس است.
- [ ] نشست‌های فعال مدیریت و لغو از طریق API امکان‌پذیر است.
- [ ] تست‌های یکپارچه مسیرهای حیاتی (ثبت‌نام، ورود، خروج، بازیابی) را پوشش می‌دهند.

## تعریف انجام کار (Definition of Done)
- [ ] مستندات API (مسیرها و اسامی رویدادها) در ویکی یا README سرویس به‌روزرسانی شده‌اند.
- [ ] پوشش تست‌های خودکار برای سناریوهای حیاتی به حداقل ۸۰٪ رسیده است.
- [ ] محیط آزمایشی با حساب نمونه به‌روزرسانی شده و نتایج تست در PR مستند شده است.
- [ ] استقرار خودکار (CI/CD) بدون خطا اجرا شده و بررسی امنیتی اولیه انجام شده است.
