# GAME CORE - منطق اصلی بازی مافیا

## خلاصه
تعریف و پیاده‌سازی چرخه‌های اصلی بازی (روز/شب)، نقش‌ها، و قوانین رأی‌گیری که تجربه مافیا را ممکن می‌سازد.

## معیارهای پذیرش
- [ ] نقش‌های مافیا، شهروند، کارآگاه، دکتر و نقش‌های قابل گسترش پشتیبانی می‌شوند.
- [ ] چرخه روز/شب با تایمر قابل تنظیم و رویدادهای مرتبط پیاده‌سازی شده است.
- [ ] رأی‌گیری، محاکمه، و اعلام نتایج مطابق قوانین مافیا عمل می‌کند.
- [ ] منطق بازی با تست‌های واحد و ترکیبی برای سناریوهای کلیدی پوشش داده می‌شود.
- [ ] مستندسازی نقش‌ها و حالات ویژه برای تیم طراحی/QA آماده است.

## تعریف انجام کار (Definition of Done)
- [ ] سناریوهای حیاتی در تست‌های شبیه‌سازی‌شده (Integration/Simulation) پوشش داده شده‌اند.
- [ ] فایل‌های پیکربندی نقش‌ها و پارامترهای بازی نسخه‌بندی و در مستندات توضیح داده شده‌اند.
- [ ] تغییرات breaking در قراردادهای API به تیم‌های مشتری اطلاع‌رسانی شده است.
- [ ] نمودار وضعیت چرخه روز/شب در اسناد طراحی به‌روزرسانی شده است.
