# LOBBY - مدیریت لابی و تطبیق بازیکنان

## خلاصه
طراحی سیستمی برای ایجاد و مدیریت لابی‌های عمومی و خصوصی، دعوت بازیکنان، و آماده‌سازی تیم‌ها پیش از شروع بازی.

## معیارهای پذیرش
- [ ] بازیکنان می‌توانند لابی عمومی یا خصوصی ایجاد کنند و تنظیمات اساسی را تعیین نمایند.
- [ ] دعوت به لابی از طریق لینک اشتراک‌گذاری یا جستجوی نام کاربری امکان‌پذیر است.
- [ ] وضعیت آماده‌باش (Ready Check) پیش از شروع بازی وجود دارد و نتیجه آن اطلاع‌رسانی می‌شود.
- [ ] وضعیت لابی (بازیکنان حاضر، دعوت‌ها، آماده‌باش) در UI به‌روزرسانی لحظه‌ای دارد.
- [ ] سناریوهای کلیدی با تست‌های انتها به انتها پوشش داده شده است.

## تعریف انجام کار (Definition of Done)
- [ ] راهنمای استفاده از لابی (ایجاد، پیوستن، دعوت) در مستندات کاربر به‌روزرسانی شده است.
- [ ] رویدادهای بلادرنگ مربوط به وضعیت لابی در Swagger یا مستندات API ثبت شده‌اند.
- [ ] متریک‌های پایه (تعداد لابی فعال و نرخ شروع بازی) در داشبورد مانیتورینگ اضافه شده‌اند.
- [ ] سناریوهای خودکار QA برای ایجاد و پیوستن به لابی در CI اجرا می‌شوند.
